Welcome To CSIS 1280!

Why Take CSIS 1280?

All programs used in this course are totally free, you will be able to make your own webpages without having to pay anything for your computer!

Summary

CSIS 1280 is a course that teaches you the basics of designing a webpage using various computer languages such as HTML, CSS, and Javascript.

What You Will Learn

In CSIS 1280, students will be able to learn how to make structured webpages using HTML and additionally using CSS to style those pages. Students will also be taught basic concepts in Javascipt to implement client-side script.
